# Cover Letter

Dear Editor,

We are pleased to submit our manuscript entitled "Agentic Engineering: A Multi-Agent LLM Framework for Solving Chemical Engineering Tasks with Open-Source Thermodynamic Software" for consideration in *Computers & Chemical Engineering*.

## Scope and novelty

This manuscript presents a structured framework that combines multiple specialized large language model (LLM) agents with a rigorous open-source thermodynamic library to solve diverse chemical engineering tasks end-to-end. Unlike prior work that addresses individual aspects of AI-assisted engineering (flowsheet construction, code generation, or simulation access), our framework integrates problem definition, computational simulation, validation, and report generation into a coherent, validated workflow. The paper is directly aligned with the journal's scope in AI/ML for chemical engineering, computational methods, process simulation, and cyberinfrastructure.

The key novelty is the demonstration that a multi-agent architecture with separation of LLM reasoning from rigorous numerical computation — where agents compose and validate code invoking a thermodynamic library rather than performing calculations themselves — provides the reliability needed for engineering-grade results. We validate this across eight real-world case studies spanning six engineering disciplines, providing the broadest demonstration of AI-assisted chemical engineering to date.

## Key findings

- A three-stage workflow (scope/research, analysis/evaluation, reporting) with validation gates provides engineering rigor for AI-assisted computation
- Separation of LLM reasoning from thermodynamic computation achieves mass balance closures below 0.02% across all case studies
- The framework generalizes across cryogenics, field development, flow assurance, equipment design, well safety, and model conversion
- Human oversight remains essential for problem formulation and decision-making; the framework augments rather than replaces engineering judgment
- The open-source foundation enables full reproducibility and community-driven improvement
